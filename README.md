## Chef Claude 

Chef Claude is an AI-powered recipe generator built with React and Vite. Users can enter ingredients they have on hand, and the application generates recipe suggestions using the Gemini API.

## Features
Add ingredients dynamically

Generate recipes based on available ingredients

AI-powered recipe suggestions

Responsive and clean user interface

Built with modern React practices
## Tech Stack
React

Vite

JavaScript (ES6+)

CSS

Gemini API
